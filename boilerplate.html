Presentation on DXF Task

What is DXF?

DXF is a file extension for a graphic image format typically used with CAD software. DXF stands for Drawing eXchange Format. It has two versions ASCII and binary forms.

Parts of DXF files

1. HEADER section – General information about the drawing. Each parameter has a variable name and an associated value.

2. CLASSES section – Holds the information for application-defined classes whose instances appear in the BLOCKS, ENTITIES, and OBJECTS sections of the database.

3. TABLES section – This section contains definitions of named items.

Types of table section :Layer (LAYER) table and User Coordinate System (UCS) table

4. BLOCKS section – This section contains Block Definition entities describing the entities comprising each Block in the drawing.

5. ENTITIES section – This section contains the drawing entities, including any Block References.

6. END OF FILE

Group Codes

The group code is an integer which implicitly defines the value type and acts as a key for the value. In ASCII DXF, group codes and values are written in a single line each.
